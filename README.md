**Lexi Bike Agencies** is a Django-based web application designed to manage day-to-day operations of a bike agency.  
The project focuses on handling employee data, maintaining records, and exposing RESTful APIs for seamless integration with frontend or third-party applications.

This application follows Django best practices and uses **Class-Based Views (CBVs)** along with **Django REST Framework (DRF)** to build scalable and maintainable backend services.
